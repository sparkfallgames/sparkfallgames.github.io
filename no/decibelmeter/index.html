Hvorfor Decibel Meter

Måling i sanntid

dB-avlesning i sanntid med MIN, GJ.SN og MAKS på ett blikk.

Daglig støydose

Følger eksponeringen din etter NIOSH-retningslinjen på 85 dB / 8 t.

Ingenting tas opp

Lyden analyseres i sanntid og lagres eller lastes aldri opp.

Widget og Siri

Start og stopp målinger fra Hjem-skjermen eller med stemmen.

Personvern
